Log Parser

This is a simple app that takes a log file and displays the top 10 users. Log lines need to match this regex:

'^\d+? \d{4}-\d{2}-\d{2} \d{2}:\d{2}:\d{2} ".+?" \d+? [\d.]+? (?P<username>.+?)\s'
